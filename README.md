# ControlBet - BetCore DB

## Descripción del proyecto

ControlBet es una startup orientada al diseño de soluciones de gestión de datos para casas de apuestas deportivas. El producto propuesto, **BetCore DB**, consiste en un modelo de gestión y monitoreo de datos que busca organizar información operativa y transaccional relacionada con usuarios, apuestas, eventos deportivos, cuotas, pagos, saldos, comprobantes y transacciones financieras.

El proyecto no tiene como objetivo desarrollar una plataforma final, sino diseñar una solución de base de datos relacional que permita mejorar la trazabilidad de las operaciones, reducir inconsistencias, facilitar el control financiero y apoyar la toma de decisiones estratégicas dentro de una casa de apuestas deportiva.

## Objetivo del proyecto

Diseñar una base de datos relacional para una casa de apuestas deportivas, considerando el análisis del dominio del problema, los requisitos identificados mediante entrevistas, las entidades principales, los atributos, las relaciones y la implementación física en SQL Server.

## Alcance

El alcance de este proyecto incluye:

- Análisis de antecedentes y problemática.
- Identificación de segmentos objetivo.
- Diseño y registro de entrevistas.
- Análisis de requisitos funcionales y no funcionales.
- Identificación de entidades y atributos.
- Diseño del modelo entidad-relación lógico.
- Diseño del modelo entidad-relación físico.
- Evaluación y elección del DBMS relacional.
- Implementación inicial del modelo en SQL Server.
